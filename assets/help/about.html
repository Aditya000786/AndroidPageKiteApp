What is PageKite?

PageKite is a tool which makes your Android phone a fully-fledged part of the World Wide Web. It lets you choose a public name for your device and makes it globally reachable whenever it is powered on and has Internet connectivity.

The app relies on access to a front-end relay server, either one provided by the pagekite.net service or one you run on your own server.

See below for some examples of server apps which are known to work with PageKite.

Getting Started

Using pagekite.net

  1. Install an SSH or Web Server app
  2. Configure the ports in Local Server Settings and the server app so the ports match.
  3. Sign up for a pagekite.net account.
  4. Configure the Account Details with your kite name and shared secret.

Running your own relay

This process is more or less the same as above, except instead of signing up for an account you will need to purchase your own domain and install the Open Source pagekite.py relay software on a computer or VPS with a public IP address.
